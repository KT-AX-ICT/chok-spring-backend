# Dashboard Domain

## Package Owner

- 최종 구조/리뷰 책임: 박연지

## Scope Ownership

| Scope | Primary | Support | Notes |
| --- | --- | --- | --- |
| 대시보드 집계 API | 박연지 | 이석진 | 저장된 로그/분석 결과 기반 집계를 제공한다. |
| Dashboard 화면 API 연결 | 박연지 | 임은아 | UiDemo mock data 제거 후 시연 첫 화면에 연결한다. |
| 집계 항목 검수 | 임은아 | 박연지 | P0 요구사항과 시연 시나리오 기준으로 누락을 확인한다. |
| 데이터 표현 체크 | 윤혜림 | 박연지 | 화면에서 이상 로그/분석 결과가 이해 가능하게 보이는지 점검한다. |

## Scope

- 메인 대시보드용 집계 API
- 최근 분석 현황, 이상 로그 수, 최근 이상 로그/분석 결과 요약
- 시간대별 또는 범위별 카운트 제공
- UiDemo 대시보드 화면의 mock data 제거 및 API 연결 지원

## Responsibilities

- 화면에서 바로 사용할 수 있는 집계 응답 모델을 제공한다.
- 분석 결과와 로그 데이터를 조합해 대시보드 지표를 만든다.
- 실시간 기능처럼 보이는 요구는 저장 데이터 기반 조회 또는 리플레이 시뮬레이션임을 명확히 구분한다.
- 시연 첫 화면에 필요한 총 로그 수, 이상 로그 수, 최근 분석 상태를 우선 제공한다.
- 별도 Alert 파트는 현재 우선순위에서 제외하거나 보류한다.

## Out Of Scope

- 원본 로그 상세 탐색 중심 화면
- FastAPI 분석 실행 로직
- 별도 Alert 서비스/알림 발송
- 실제 운영 시스템 직접 모니터링
