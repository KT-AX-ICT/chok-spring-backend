# Claude Code Instructions

Before working in this repository, check whether `CLAUDE.local.md` exists. If it exists, read it first and treat it as the local source of machine-specific paths and personal instructions.

For work-in-progress context (decisions, open issues, next tasks), read the **top** of `docs/worklog.md` first if it exists — the **TL;DR** and **Decision Log** sections (near the top) give current state, next tasks, and key decisions in only ~3-4k tokens. Don't read the whole file by default: older dated logs are split into `docs/worklog-archive.md` and are reference-only — open them only when you need detail on a specific past change. Note: `docs/` is gitignored (local-only), so do not commit it.

Do not commit or upload local instruction files such as `AGENTS.local.md` or `CLAUDE.local.md`.
